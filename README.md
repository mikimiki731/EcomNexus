# EcomNexus Landing Page

A modern, conversion-focused landing page for EcomNexus, an e-commerce website development agency.

## Features
- Responsive design for all devices
- Dynamic service packages
- Accessible navigation and forms
- SEO and social meta tags
- Ready for GitHub Pages deployment

## Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-github-username/your-repo-name.git
   ```
2. **Add your favicon:**
   Place your `favicon.ico` in the root directory.
3. **Customize:**
   - Update meta tags in `index.html` for your brand and deployment URL.
   - Update contact and social links as needed.

## Deployment (GitHub Pages)
1. Push your code to the `main` branch of your repository.
2. Go to **Settings > Pages** in your GitHub repo.
3. Set the source to the `main` branch and `/ (root)` folder.
4. Save and wait for your site to be published at:
   `https://your-github-username.github.io/your-repo-name/`
